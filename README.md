# 地貌实习可视化Web3D平台

基于Cesium的地貌实习可视化Web3D平台，为地质、地理、环境科学等专业的学生和研究人员提供沉浸式的野外实习体验和数据记录工具。

## 功能特性

### ✅ 已实现功能（MVP版本）

- **三维地球显示**: 基于Cesium的高质量3D地球渲染
- **高精度地形**: Cesium World Terrain提供30米分辨率地形数据
- **多种底图**: 支持卫星影像、道路地图等多种图层选择
- **轨迹数据管理**: 支持GPX和JSON格式的轨迹文件导入/导出
- **轨迹颜色调整**: 支持实时调整轨迹颜色，提供预设颜色选择
- **文件系统集成**: 支持现代浏览器的File System Access API
- **示例数据**: 内置大同地区地貌观察点数据
- **轨迹可视化**: 实时轨迹显示和交互，自动贴合地形
- **数据导出**: 支持轨迹数据的GPX格式导出

### 🚧 开发中功能

- **实时GPS轨迹记录**
- **笔记系统**（文字、图片、语音）
- **DEM数据处理**
- **地形分析工具**

## ✅ Cesium配置说明

应用已配置Cesium Ion访问令牌，可以直接使用所有高级功能：

- ✅ 高精度Cesium World Terrain地形数据
- ✅ 多种卫星影像和地图图层  
- ✅ 图层选择器（BaseLayerPicker）
- ✅ 高质量3D地形渲染

详细配置信息请参考 [CESIUM_SETUP.md](./CESIUM_SETUP.md)。

如果遇到任何问题，请参考 [TROUBLESHOOTING.md](./TROUBLESHOOTING.md) 故障排除指南。

### 可用的高级功能

**高精度地形数据**:
- Cesium World Terrain全球地形数据，分辨率可达30米
- 真实的地形阴影和光照效果
- 轨迹和标记自动贴合地形表面

**图层选择器**:
在应用右上角的图层选择器中，您可以选择：
- Bing Maps Aerial（高分辨率卫星影像）
- Bing Maps Road（道路地图）
- OpenStreetMap（开源地图）
- Natural Earth II（自然地球影像）
- ESRI World Imagery（ESRI世界影像）

## 技术栈

- **前端框架**: React 18 + TypeScript + Vite
- **3D引擎**: Cesium.js
- **UI组件**: Ant Design
- **状态管理**: Zustand
- **文件处理**: File System Access API + FileSaver.js

## 使用指南

### 1. 加载示例数据

点击左侧边栏的"加载示例数据"按钮，系统会自动加载大同地区的地貌观察点数据，包括：
- 地貌观察点（红色标记）
- 实习路径（蓝色轨迹）

### 2. 导入轨迹文件

- 点击"导入轨迹"按钮
- 选择GPX或JSON格式的轨迹文件
- 系统会自动解析并在地图上显示

### 3. 文件系统集成（Chrome等现代浏览器）

- 点击"选择工作目录"
- 选择本地文件夹作为工作目录
- 系统会自动扫描并加载目录中的轨迹文件
- 支持轨迹数据的本地保存

### 4. 轨迹管理

- 点击"轨迹管理"打开轨迹列表
- 支持轨迹的显示/隐藏切换
- 支持飞行到指定轨迹
- 支持轨迹删除和导出

### 5. 轨迹颜色调整

- 在轨迹管理面板中，每个轨迹卡片都有颜色选择器
- 点击颜色选择器可以打开颜色面板
- 支持自定义颜色选择和预设颜色快速选择
- 颜色更改会实时反映在3D地图上
- 侧边栏轨迹列表中也显示颜色指示器

## 数据格式

### GPX格式
标准GPX 1.1格式，支持轨迹点的经纬度、高程和时间信息。

### JSON格式
```json
{
  "id": "track_id",
  "name": "轨迹名称",
  "points": [
    {
      "lng": 113.677983,
      "lat": 40.105708,
      "elevation": 1227,
      "time": "2024-01-15T09:00:00.000Z"
    }
  ],
  "color": "#ff0000",
  "visible": true,
  "createdAt": "2024-01-15T08:00:00.000Z"
}
```

### 观察点坐标格式
```
经度,纬度,高程,编号
113.677983,40.105708,1227,1
113.677172,40.106972,1228,2
```
