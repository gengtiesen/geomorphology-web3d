# 地貌实习Web3D可视化平台

JavaScript与Web开发课程大作业。

## 👉 在线访问

**https://gengtiesen.github.io/geomorphology-web3d/**

点击即看，无需安装任何东西。

## 项目简介

将2025年大同野外地质地貌实习数据（10天、39个观测点的GPS轨迹、剖面图、实地照片）整合到3D地球引擎中，实现轨迹渲染、观测点交互查询、高程剖面分析和鹰眼图等功能。

## 技术栈

JavaScript（ES6+） · React 18 · Vite · Cesium.js · Ant Design · Zustand · ECharts · Canvas 2D API

## 本地运行（可选）

```bash
npm install
npm run dev
```
