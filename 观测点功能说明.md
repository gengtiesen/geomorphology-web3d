# 观测点功能说明

## 功能概述

地貌Web3D平台现在支持观测点的详细信息查看功能，包括：

1. **知识点内容** - 显示观测点相关的地质知识
2. **剖面图** - 展示地质剖面图像
3. **实地图片** - 显示观测点的实地拍摄照片

## 使用方法

### 1. 导入观测点数据

- 在侧边栏点击"导入观察点"按钮
- 选择包含观测点坐标的文本文件（格式：经度,纬度,海拔,编号）
- 系统会自动在地图上显示观测点标记

### 2. 查看观测点详情

有两种方式查看观测点详情：

#### 方式一：点击地图上的观测点
- 直接在地图上点击红色的观测点标记
- 系统会自动弹出详情对话框

#### 方式二：通过侧边栏
- 在侧边栏的"观察点"列表中点击任意观测点卡片
- 或点击观测点卡片上的信息按钮

### 3. 观测点详情内容

详情对话框包含以下信息：

- **基本信息**：名称、经纬度、海拔、描述等
- **知识点**：相关地质知识和科学原理
- **剖面图**：地质剖面图像，可放大查看
- **实地图片**：观测点的现场照片，支持图片预览

## 文件组织结构

观测点的详细信息文件需要按以下命名规则放置在 `public/观测点文档/` 目录下：

```
public/观测点文档/
├── 观测点1 知识点.txt      # 知识点内容文本文件
├── 观测点1 剖面图.jpg      # 剖面图图像文件
├── 观测点1 图片.jpg        # 实地图片文件
├── 观测点2 知识点.txt
├── 观测点2 剖面图.jpg
└── 观测点2 图片.jpg
```

## 示例数据

项目包含了观测点1的示例数据：

- **知识点内容**：包含中国黄土地层的详细科学知识
- **剖面图**：展示地质剖面结构
- **实地图片**：观测点的现场照片

## 技术实现

- 使用 React + TypeScript 开发
- 集成 Cesium 3D地图引擎
- 支持异步加载观测点详细信息
- 使用 Ant Design 组件库提供良好的用户界面
- 支持图片预览和放大功能

## 注意事项

1. 观测点详细信息文件的命名必须与观测点名称完全匹配
2. 支持的图片格式：JPG、PNG
3. 知识点文件使用UTF-8编码的文本格式
4. 文件需要放置在 `public/观测点文档/` 目录下才能被正确访问
