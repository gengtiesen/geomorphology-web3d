## 使用方法

### 1. 启动应用
```bash
npm run dev
```

### 2. 加载示例数据
- 点击"加载示例数据"按钮
- 地图上会显示观测点标记

### 3. 查看观测点详情
- **方式一**：直接点击地图上的红色观测点标记
- **方式二**：在侧边栏点击观测点卡片

### 4. 详情内容
- **基本信息**：坐标、海拔、描述等
- **知识点**：地质相关知识内容
- **剖面图**：地质剖面图像
- **实地图片**：观测点现场照片

## 文件命名规则

观测点详细信息文件必须按以下格式命名：
- 知识点：`{观测点名称} 知识点.txt`
- 剖面图：`{观测点名称} 剖面图.jpg`
- 实地图片：`{观测点名称} 图片.jpg`
